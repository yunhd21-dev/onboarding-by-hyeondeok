# ToDo List App
## Framework
* [Webpack](https://webpack.kr/)
* [ESlint](https://eslint.org/)
* [Prettier](https://prettier.io/)
* [Storybook](https://storybook.js.org/)
* [Jest](https://jestjs.io/)
* [Cypress](https://www.cypress.io/)

## ToDo List App 실행
```
npm run start
```

## Storybook 실행
```
npm run storybook
```

## Jest(단위 테스트) 실행
```
npm run jest
```

## Cypress(E2E 테스트) 실행
```
npm run cy:open
```
